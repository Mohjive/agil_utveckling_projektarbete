WCAG

Web Content Accessibility Guidelines är riktlinjer för hur webbplatser tillgänglighetsanpassas för att säkerställa att alla (eller så nära alla som möjligt) ska få ta del av hela webbplasten.

Videoklippet som ligger här intill (eller det på sidan om projektmetodik) är som exempel inte alls anpassat för att vara tillgängligt. Videon innehåller länkar på olika ställen på skärmen, som när man tabbar sig mellan dem, kan vara svåra att förstå vad de gör om man t ex saknar syn. Många användare använder så kallade skärmläsare som läser upp innehållet. Det finns ju dock inte några hundraprocentiga sätt att "läsa upp" en bild eller ännu värre; en video, långt ifrån faktiskt. Det innebär att om utvecklaren inte tar hänsyn till detta och lägger till information som skärmläsare kan läsa upp så går användaren miste om innehåll. När du är på den här hemsidan kan du använda dig av tabbtangenten för att hoppa mellan länkar på sidan. Den första länken som dyker upp om du tabbar är en så kallad skip-to-main-länk. Den är inte synlig för ögat innan den aktiveras då den kodats att ligga utanför bild fram till dess att användaren tabbar till den.
