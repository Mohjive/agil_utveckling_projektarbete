Html - Hemsidans skelett

Html står för hyper text markup language. Det är vad som beskriver strukturen på en hemsida. Med hjälp av så kallade taggar definieras innehållet och kan utifrån definition/taggar visa text, bilder osv på det sätt som önskas. Texten i det här stycker är skrivet inom paragraf-taggar och bilden på de tre kropparna ligger innanför image-taggar. Menyn längst upp på sidan är i koden omgiven av taggar som definierar en header.

JS - Hemsidans muskler

Javaskript är ett objektorienterat scriptspråk som används för att bygga avancerade applikationer och webbplatser. Det är tack vare javaskript som t.ex kontaktformuläret har funktion. Om det är med hjälp av javaskript som menyn längst upp på sidan kan displayas som en lång rad om du använder en stor skärm och en dropdown-meny om du använder t.ex en telefon.

CSS - Hemsidans hud, kläder och smink

Css står för cascading style sheets och är pråket som används för att style'a html-dokument, hemsidans utsida. Med hjälp av css beskrivs hur html-elementen ska presenteras visuellt i webbläsaren. Det är i css-koden det bestämts att länkarna på den här hemsidan är lila.
