Html - Hemsidans skelett

Html står för hyper text markup language. Det är vad som beskriver strukturen på en hemsida. Med hjälp av så kallade taggar definieras innehållet och kan utifrån definition/taggar visa text, bilder osv på det sätt som önskas.

JS - Hemsidans muskler

Javaskript är ett objektorienterat scriptspråk som används för att bygga avancerade applikationer och webbplatser. Det är tack vare javaskript som kontaktformuläret har funktion.

CSS - Hemsidans hud, kläder och smink

Css står för cascading style sheets och är pråket som används för att style'a html-dokument, hemsidans utsida. Med hjälp av css beskrivs hur html-elementen ska presenteras visuellt i webbläsaren.
